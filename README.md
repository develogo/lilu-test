
# Lilu Test



## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

1 - Go to the project directory and open api folder on ItelliJ or Eclipse an Run

 2 - Go to web project

```bash
  cd liluapp
```

3 - Get flutter dependeces

```bash
  flutter pub get
```

4 - Run the project

```bash
  flutter run -d chrome
```
